# Ejercicio 02 — Primeros Tests Unitarios en Java

> **Semana 05 · Practicas · Ejercicio 02** | Duracion estimada: 2 h

---

## Objetivo

Escribir una suite inicial en JUnit 5 aplicando:

- Patron AAA
- Assertions basicas
- `assertThrows` para validaciones
- `@DisplayName` para legibilidad

---

## Instrucciones

### Paso 1 — Abrir tests starter

Abre `starter/src/test/java/com/bootcamp/UserUtilsTest.java`.

### Paso 2 — Descomentar por secciones

Sigue los bloques `PASO 1`, `PASO 2` y `PASO 3`.

### Paso 3 — Ejecutar tests

```bash
mvn test
```

### Paso 4 — Completar assertions faltantes

Ajusta valores esperados y valida excepciones donde corresponda.

---

## Resultado esperado

- Minimo 6 tests en verde
- Happy path + casos invalidos
- Estructura AAA consistente
