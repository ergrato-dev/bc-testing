# Ejercicio 02 - pytest-mock, Spy y Stub

## Objetivo

Usar `mocker` para crear stubs, espiar metodos y validar interacciones con bajo acoplamiento.

## Tiempo estimado

90 minutos.

## Paso a paso

### Paso 1: Stub con return_value

Abre `starter/test_invoice_service.py` y descomenta PASO 1.

### Paso 2: Spy sobre metodo real

Descomenta PASO 2 para observar llamadas sin reemplazar implementacion.

### Paso 3: Revisar solucion

Compara con `solution/test_invoice_service.py`.

## Comandos sugeridos

```bash
pytest -q
pytest -k invoice
```
