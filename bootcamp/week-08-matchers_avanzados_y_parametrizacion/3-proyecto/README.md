# Proyecto Semana 08 - Suite Parametrizada con Jest

## Objetivo

Construir una suite de tests del dominio asignado usando matchers adecuados y parametrizacion con `test.each`.

## Entregable obligatorio

- Suite en JavaScript/Jest.
- Minimo 10 tests ejecutables.
- Al menos 2 bloques con `test.each`.

## Reglas del proyecto

1. Usa solo JavaScript y Jest.
2. Mantiene nomenclatura tecnica en ingles.
3. Usa comentarios y explicaciones en espanol en la documentacion de entrega.
4. No uses dominio de ejemplo prohibido por politica anticopia.

## Requisitos tecnicos minimos

- Casos validos, invalidos y limite.
- Matchers especificos y legibles.
- Parametrizacion con tablas de casos.
- Sin servicios externos reales.

## Estructura starter

- `starter/item-pricing.service.test.js`

> La carpeta `solution/` del proyecto se mantiene fuera del repositorio publico.
