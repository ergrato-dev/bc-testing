# Ejercicio 02 - Parametrizacion con `test.each`

## Objetivo

Reducir duplicacion de tests mediante tablas de casos claras y mantenibles.

## Tiempo estimado

90 minutos.

## Pasos

1. Abre `starter/discount.service.test.js`.
2. Descomenta PASO 1 para casos simples.
3. Descomenta PASO 2 para tabla con `test.each`.
4. Descomenta PASO 3 para casos invalidos.
5. Compara con `solution/discount.service.test.js`.

## Comando sugerido

```bash
yarn test discount.service.test.js
```
