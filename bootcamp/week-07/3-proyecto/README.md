# Proyecto Semana 07 - Suite Modular con Jest

## Objetivo

Construir una suite de tests unitarios en JavaScript con estructura modular, hooks bien aplicados y uso basico de dobles de prueba.

## Entregable obligatorio

- Suite en Jest del dominio asignado.
- Minimo 10 tests automatizados ejecutables.
- Evidencia de al menos 3 tests que usan mock/stub/spy de forma justificada.

## Reglas del proyecto

1. Usa exclusivamente JavaScript y Jest en esta semana.
2. Adapta el starter a tu dominio asignado.
3. Mantiene nomenclatura tecnica en ingles.
4. Documenta decisiones en espanol en tu entrega.

## Requisitos tecnicos minimos

- Organizacion con `describe` principal y subbloques por metodo.
- Hooks (`beforeEach`, `afterEach`) solo donde aporten claridad.
- Casos: happy path, invalidaciones y edge cases.
- Sin acceso a BD ni APIs reales.

## Estructura starter

- `starter/item.service.test.js`

> La carpeta `solution/` del proyecto se mantiene fuera del repositorio publico.
