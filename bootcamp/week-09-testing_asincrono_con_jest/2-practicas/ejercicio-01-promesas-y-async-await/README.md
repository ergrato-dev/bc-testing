# Ejercicio 01 - Promesas y Async/Await

## Objetivo

Escribir tests asincronos confiables con `async/await`, `resolves` y `rejects`.

## Tiempo estimado

90 minutos.

## Pasos

1. Abre `starter/user.repository.test.js`.
2. Descomenta PASO 1 para caso exitoso con `await`.
3. Descomenta PASO 2 para `resolves`.
4. Descomenta PASO 3 para `rejects`.
5. Compara con `solution/user.repository.test.js`.

## Comando sugerido

```bash
yarn test user.repository.test.js
```
