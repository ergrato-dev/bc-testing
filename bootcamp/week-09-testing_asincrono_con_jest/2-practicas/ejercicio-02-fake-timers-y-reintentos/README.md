# Ejercicio 02 - Fake Timers y Reintentos

## Objetivo

Controlar el tiempo de ejecucion en tests de retry usando fake timers.

## Tiempo estimado

90 minutos.

## Pasos

1. Abre `starter/retry.service.test.js`.
2. Descomenta PASO 1 para activar fake timers.
3. Descomenta PASO 2 para simular paso del tiempo.
4. Descomenta PASO 3 para validar agotamiento de reintentos.
5. Compara con `solution/retry.service.test.js`.

## Comando sugerido

```bash
yarn test retry.service.test.js
```
