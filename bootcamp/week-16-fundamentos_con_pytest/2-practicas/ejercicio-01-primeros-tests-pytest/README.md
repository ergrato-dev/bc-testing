# Ejercicio 01 - Primeros Tests con pytest

## Objetivo

Construir una primera suite basica en Python usando `pytest` y patron AAA.

## Tiempo estimado

90 minutos.

## Paso a paso

### Paso 1: Caso base

Abre `starter/test_pricing_rules.py` y descomenta PASO 1.

### Paso 2: Regla de descuento

Descomenta PASO 2 para validar descuento de cliente premium.

### Paso 3: Error esperado

Descomenta PASO 3 para validar excepcion en monto invalido.

### Paso 4: Comparar con solucion

Revisa `solution/test_pricing_rules.py`.

## Comando sugerido

```bash
pytest -q
```
