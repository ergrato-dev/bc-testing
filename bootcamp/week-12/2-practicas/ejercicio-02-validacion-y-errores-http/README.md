# Ejercicio 02 - Validacion y Errores HTTP

## Objetivo

Validar respuestas de error y contratos JSON consistentes en endpoints REST.

## Tiempo estimado

90 minutos.

## Paso a paso

### Paso 1: Caso de validacion 400

Abre `starter/app-errors.test.js` y descomenta PASO 1.

### Paso 2: Caso de no encontrado 404

Descomenta PASO 2 y valida payload de error.

### Paso 3: Caso de conflicto 409

Descomenta PASO 3 y verifica respuesta.

### Paso 4: Revisar solucion

Compara con `solution/app-errors.test.js`.

## Comando sugerido

```bash
yarn test app-errors.test.js
```
