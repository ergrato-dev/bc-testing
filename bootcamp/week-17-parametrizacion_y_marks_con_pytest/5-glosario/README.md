# Glosario Semana 17 - Parametrizacion y Marks

## I

- **id de caso**: etiqueta legible para identificar una fila de `parametrize`.

## M

- **mark**: etiqueta de pytest para clasificar pruebas por objetivo o costo.

## P

- **parametrize**: mecanismo para ejecutar un mismo test con multiples entradas.

## R

- **regression suite**: conjunto amplio de pruebas para detectar regresiones funcionales.

## S

- **smoke suite**: conjunto minimo de pruebas criticas para feedback rapido.
