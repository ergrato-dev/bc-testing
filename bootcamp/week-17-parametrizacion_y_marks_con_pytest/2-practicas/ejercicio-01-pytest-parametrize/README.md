# Ejercicio 01 - Parametrize con Casos de Negocio

## Objetivo

Aplicar `@pytest.mark.parametrize` para cubrir casos de comportamiento, borde y error sin duplicar tests.

## Tiempo estimado

90 minutos.

## Paso a paso

### Paso 1: Caso base parametrizado

Abre `starter/test_shipping_rules.py` y descomenta PASO 1.

### Paso 2: Agregar caso borde

Descomenta PASO 2 para validar limite de envio gratis.

### Paso 3: Parametrizar error esperado

Descomenta PASO 3 para entradas invalidas.

### Paso 4: Revisar solucion

Compara con `solution/test_shipping_rules.py`.

## Comando sugerido

```bash
pytest -q
```
