# Ejercicio 01 - Organizacion de Suite Jest

## Objetivo

Reestructurar una suite plana en una suite organizada por comportamiento usando `describe` anidados.

## Tiempo estimado

90 minutos.

## Pasos

1. Abre `starter/order.service.test.js`.
2. Descomenta PASO 1 para crear el bloque principal.
3. Descomenta PASO 2 para organizar por metodos.
4. Descomenta PASO 3 para agregar nomenclatura profesional.
5. Compara con `solution/order.service.test.js`.

## Comando sugerido

```bash
yarn test order.service.test.js
```
