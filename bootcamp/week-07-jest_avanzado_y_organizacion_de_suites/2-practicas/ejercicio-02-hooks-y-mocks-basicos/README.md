# Ejercicio 02 - Hooks y Mocks Basicos

## Objetivo

Aplicar hooks de ciclo de vida y dobles de prueba simples para aislar dependencias.

## Tiempo estimado

90 minutos.

## Pasos

1. Abre `starter/tests/notification.service.test.js`.
2. Descomenta PASO 1 para inicializacion con `beforeEach`.
3. Descomenta PASO 2 para stub de respuesta.
4. Descomenta PASO 3 para spy de colaborador.
5. Compara con `solution/tests/notification.service.test.js`.

## Comando sugerido

```bash
yarn test notification.service.test.js
```
