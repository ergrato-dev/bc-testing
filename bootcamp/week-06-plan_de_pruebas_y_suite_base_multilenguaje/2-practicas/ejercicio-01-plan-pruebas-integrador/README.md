# Ejercicio 01 - Plan de Pruebas Integrador

## Objetivo

Diseñar un plan de pruebas breve pero completo para un modulo pequeno del dominio asignado.

## Tiempo estimado

75 minutos.

## Instrucciones

1. Abre `starter/test-plan-template.md`.
2. Completa cada seccion en orden.
3. Usa IDs de requerimiento (`REQ-001`, `REQ-002`, etc.).
4. Prioriza casos por riesgo.
5. Compara tu trabajo con `solution/test-plan-template.md`.

## Resultado esperado

Un plan de pruebas con:

- alcance y exclusiones,
- riesgos,
- al menos 8 casos,
- criterios de entrada/salida,
- matriz de trazabilidad.
