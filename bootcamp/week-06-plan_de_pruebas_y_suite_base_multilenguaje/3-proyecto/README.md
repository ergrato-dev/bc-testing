# Proyecto Semana 06 - Integrador de Fundamentos

## Objetivo

Construir una suite base de calidad para tu dominio asignado, con trazabilidad desde requerimientos hasta tests.

## Entregable obligatorio

Una suite de pruebas en **un lenguaje principal** (JS, Python o Java) y evidencia de al menos 3 casos equivalentes en un segundo lenguaje.

## Reglas del proyecto

1. Usa tu dominio asignado por el instructor.
2. No copies ejemplos de otros aprendices.
3. Manten nomenclatura tecnica en ingles.
4. Explica decisiones en espanol dentro del README de tu entrega.

## Requisitos minimos

- Minimo 10 tests automatizados en tu lenguaje principal.
- Minimo 3 tests equivalentes en un segundo lenguaje.
- Matriz de trazabilidad (`Requirement ID -> Test Case ID -> test name`).
- Cobertura de happy path + validaciones + casos limite.

## Estructura starter

- `starter/javascript/ItemService.test.js`
- `starter/python/test_item_service.py`
- `starter/java/src/test/java/com/bootcamp/ItemServiceTest.java`

> La carpeta `solution/` del proyecto se mantiene fuera de versionado por politica del bootcamp.
