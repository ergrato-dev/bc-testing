# Ejercicio 02 - Deteccion de Tests Fragiles

## Objetivo

Reconocer patrones de fragilidad y convertir tests debiles en tests estables y utiles.

## Tiempo estimado

90 minutos.

## Paso a paso

### Paso 1: Identificar assert debil

Abre `starter/report.builder.test.js` y descomenta PASO 1.

### Paso 2: Reemplazar assert ambiguo por contrato claro

Descomenta PASO 2 para validar estructura y campos importantes.

### Paso 3: Probar ruta de validacion

Descomenta PASO 3 para cubrir error de entrada.

### Paso 4: Verificar solucion final

Compara con `solution/report.builder.test.js`.

## Comando sugerido

```bash
yarn test report.builder.test.js --coverage
```
