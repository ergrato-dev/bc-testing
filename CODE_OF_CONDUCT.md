# Código de Conducta

## Nuestro compromiso

Como participantes y mantenedores de este proyecto, nos comprometemos a crear
un ambiente abierto, respetuoso e inclusivo para todas las personas.

## Comportamiento esperado

- Usar lenguaje respetuoso y constructivo.
- Aceptar feedback técnico con actitud colaborativa.
- Priorizar el aprendizaje y la calidad del contenido.
- Ser empáticos con distintos niveles de experiencia.

## Comportamiento inaceptable

- Acoso, insultos o descalificaciones personales.
- Comentarios discriminatorios o degradantes.
- Publicación de información privada sin consentimiento.
- Cualquier conducta intimidatoria o no profesional.

## Aplicación

Las personas mantenedoras pueden tomar acciones correctivas ante conductas que
violen este código, incluyendo advertencias, rechazo de contribuciones o
restricción de participación.

## Reporte

Si detectas una situación de incumplimiento, abre un issue privado o contacta a
las personas mantenedoras del repositorio.

## Referencia

Este documento se inspira en los principios de Contributor Covenant.
